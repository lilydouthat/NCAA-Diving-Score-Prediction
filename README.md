Project Overview

This project analyzes NCAA women’s diving performance data to identify factors associated with qualification scores and develop predictive models for future competition performance.

Qualification scores determine whether athletes advance to NCAA competition. By examining historical diving results, this analysis explores patterns in athlete performance and investigates whether future qualification outcomes can be predicted using previous competition data.

Objectives

* Analyze historical NCAA diving score data
* Identify characteristics associated with qualifying athletes
* Compare qualifying and non-qualifying performances
* Develop predictive models for qualification scores
* Generate insights for athletes and coaches

Methods

* Data Cleaning
* Exploratory Data Analysis
* Statistical Modeling
* Predictive Analytics
* Data Visualization

Data Processing

Key preprocessing steps included:

* Filtering data by diving event
* Creating qualification outcome variables
* Comparing qualifying and non-qualifying athletes
* Identifying and removing implausible observations
* Preparing datasets for statistical analysis

Skills Demonstrated

* R Programming
* Data Cleaning
* Predictive Modeling
* Statistical Analysis
* Data Visualization
* Research Communication

Project Outcomes

This project demonstrates how historical performance data can be leveraged to understand qualification trends and support evidence-based decision-making in collegiate athletics.
